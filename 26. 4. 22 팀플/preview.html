2조 NAT 포함 IP 계획표

기준: 기존 1.2.x.x는 공인 역할 IP로 유지하고, 실제 Client/Server VM에는 192.168.x.x 사설 IP를 설정한다.

핵심: 각 라우터의 LAN 쪽은 ip nat inside, Transit-SW 쪽은 ip nat outside로 본다.

전체 구조

A-PC / EDGE-DNS-DHCP C-PC / APP + Transit-SW B-PC / DATA-MGMT 192.168.1.0/24 192.168.2.0/24 192.168.3.0/24 | | | A-R ---- 1.2.123.0/24 Transit LAN ---- C-R ---- APP LAN | | +---------------------- Transit-SW ---------------------- B-R
